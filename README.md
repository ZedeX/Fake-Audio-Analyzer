### 📊 分析指标

- **采样率**：音频文件的原始采样率
- **最高有效频率**：实际包含音频信号的最高频率
- **频率截止特征**：判断是否自然衰减
- **能量分布**：分析高频区域能量变化

## 技术原理

本项目基于文章《用频谱分析找出我存的那些假无损，假高解析度音乐》的核心思路：

1. **频谱分析**：使用FFT（快速傅里叶变换）将音频从时域转换到频域
2. **STFT计算**：通过短时傅里叶变换生成时频谱图
3. **特征提取**：分析高频区域的能量分布和截止特征
4. **模式识别**：识别不同编码格式的频谱指纹

### 检测算法

```python
# MP3转码检测逻辑
if 16-20kHz平均能量 < 最大能量 * 0.001:
    判定为假无损（MP3转码特征）

# 升频检测逻辑
if 采样率 > 48kHz and 最高有效频率 < 30kHz:
    判定为假高解析度（升频）

# AAC特征检测
if 高频区域能量变化变异系数 > 2.0:
    判定为AAC编码特征
```

## 使用方法

### Web界面操作

1. **上传音频文件**
   - 点击上传区域选择文件
   - 或拖拽音频文件到上传区域
   - 支持FLAC、MP3、WAV等常见格式

2. **开始分析**
   - 点击"开始分析"按钮
   - 等待分析完成（通常10-30秒，高精度分析）

3. **查看结果**
   - **频谱图**：高质量频谱可视化（英文标签）
   - **检测报告**：详细的分析结果
   - **置信度评分**：检测结果的可信度（0-100%）
   - **问题提示**：列出检测到的异常特征
   - **详细指标**：滚降频率、频谱平坦度等专业指标

### 结果解读

#### ✅ 真无损音频特征
- 16kHz以上频率自然衰减
- 无明显的能量截断
- 高频区域平滑过渡

#### ❌ 假无损音频特征（MP3转码）
- 16kHz处有明显截断
- 16kHz以上能量极低
- 频谱图出现"一刀切"现象

#### ✅ 真高解析度音频特征
- 有效频率达到30kHz以上
- 高频截止自然平滑
- 无人工升频边界

#### ❌ 假高解析度音频特征（升频）
- 21kHz或24kHz有明显边界
- 边界以上为静音或噪音
- 有效频率远低于采样率标准

## 检测案例（优化后）

### 案例1：MP3转码的假FLAC
```
文件格式：FLAC (44.1kHz/16bit)
检测结果：假无损（置信度：92%）
问题：
  ⚠️ MP3转码特征：16kHz处能量下降97%（断崖式截断）
  ⚠️ 频谱过早滚降：在15.8kHz处能量已低于0.1%
频谱特征：在16kHz处出现明显"悬崖式"截止
检测指标：
  - 截断比例：0.023（正常>0.5）
  - 滚降频率：15.8kHz（正常>18kHz）
```

### 案例2：CD升频的假96kHz
```
文件格式：FLAC (96kHz/24bit)
检测结果：假高解析度（置信度：87%）
问题：
  ❌ 疑似升频：有效频率仅21.3kHz（应>30kHz）
  ⚠️ 21-24kHz边界后高频噪音异常
  ⚠️ 高频噪音异常：30kHz处噪音过高
频谱特征：21kHz以上出现噪音和静音混合
检测指标：
  - 有效频率：21.3kHz（真高解析度应>30kHz）
  - 边界噪音：超出正常范围3.2倍
```

### 案例3：真无损音频
```
文件格式：FLAC (44.1kHz/16bit)
检测结果：真无损（置信度：78%）
特征：16-20kHz区域有自然衰减，无截断现象
频谱特征：高频平滑过渡，能量自然降低
检测指标：
  - 截断比例：0.68（正常）
  - 滚降频率：19.2kHz（正常）
  - 频谱平坦度：0.15（正常）
```

### 案例4：真高解析度音频
```
文件格式：FLAC (96kHz/24bit)
检测结果：真高解析度（置信度：85%）
特征：有效频率达到38kHz，高频截止自然
检测指标：
  - 有效频率：38.5kHz（达标）
  - 30kHz噪音水平：正常
  - 边界检测：无明显人工边界
```

## 已知限制

1. **DSD格式**：由于DSD到PCM转换会加入滤波器，当前方法不适用
2. **专业伪造**：经过精心处理的假音频可能难以检测
3. **母带差异**：不同录音设备的频响特性可能影响判断
4. **计算资源**：大文件分析需要较多内存和CPU
